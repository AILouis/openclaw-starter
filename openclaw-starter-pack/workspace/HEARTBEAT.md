# Heartbeat Checklist

Run through these quickly. Reply HEARTBEAT_OK if nothing needs attention.

## Check (rotate, don't do all every time):
- Agent health — any sub-agents with errors or stalled tasks?
- Memory maintenance — any recent daily files worth distilling into MEMORY.md?
- Pending tasks — anything waiting for review?
- Disk/system health — anything unusual?

## Life Admin (rotate, 1-2x per day):
- Upcoming deadlines — anything the user mentioned that's time-sensitive?
- If the user mentioned something to do "later" or "tomorrow", check if it's been done.

## Proactive (only during waking hours):
- If the user hasn't been heard from in 8+ hours, send a gentle check-in
- If something important happened that the user should know about, flag it

## Monthly (1st of each month):
- Review daily memory files older than 30 days
- Distill any remaining insights into MEMORY.md
- Move raw files to `memory/archive/YYYY-MM/` (create directory if needed)

## Silent hours:
- HEARTBEAT_OK unless truly urgent
